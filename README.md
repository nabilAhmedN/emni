এই মডিউলে যা যা দেখানো হয়েছে, তার উপর ভিত্তি করে আপনাদের জন্য এই এসাইনমেন্ট তৈরি করা হয়েছে। এই এসাইনমেন্ট-এর জন্য আমরা পূর্বের মত HTML/Tailwind CSS ব্যবহার করে একটি template তৈরি করে দিয়েছি। আপনাদেরকে এই template ব্যবহার করেই এসাইনমেন্টটি করতে হবে । মনে রাখবেন, এখন পর্যন্ত যা যা দেখানো বা শেখানো হয়েছে সেগুলোর মধ্যে থেকেই আপনাকে এসাইনমেন্ট করতে হবে ।

সেই সাথে একটি 'backend' - নামক ফোল্ডার দেয়া হয়েছে। সেখানে কিছু API রয়েছে, সেগুলো ব্যবহার করে আপনাদের এই এসাইনমেন্ট এটি করতে হবে । এসাইনমেন্ট এর Installation Process এবং Documentation, backend ফোল্ডারের README.md ফাইলে দেয়া আছে ।

## এসাইনমেন্টে আপনাকে যা যা করতে হবেঃ

✓ আপনাকে আমাদের দেয়া API থেকে প্রোডাক্টস এর ডেটা নিয়ে এসে, প্রোডাক্টস লিস্ট রেন্ডার করতে হবে । এজন্য আপনাকে আমাদের দেয়া API এর ডকুমেন্টেশন পড়ে, বুঝে API ব্যবহার করতে হবে। আপনি একটি নতুন ডকুমেন্টেশন পড়ে বুঝতে পারছেন কিনা সেটি বুঝাও আমাদের এই এসাইনমেন্টের একটি উদ্দেশ্য।

✓ API থেকে নিয়ে আসা প্রোডাক্ট গুলোতে Sorting করার ফিচার Implement করতে হবে । সর্টিং এর ক্ষেত্রে Newest, Oldest, Price এর ভিত্তিতে Sort করতে হবে । এই সর্টিং এর জন্যে কোনো API নেই, এটি আপনাদের Client Side এই করতে হবে ।

✓ আমাদের দেয়া API থেকে ক্যাটাগরি লিস্ট নিয়ে এসে ফিল্টার এর অপশন গুলো রেন্ডার করতে হবে ।

✓ আপনাকে Filter ফিচার Implement করতে হবে । প্রোডাক্ট লিস্ট এর বাম পাশের সাইডবার এ Filter করার কিছু অপশন রয়েছে । যেমনঃ Category, Price, Rating এগুলোর উপড়ে ফিল্টার করতে হবে । ফিল্টার এর জন্যে API বানিয়ে দেয়া হয়েছে । আপনাকে সেই API ব্যবহার করে সার্ভার সাইড Filtering করতে হবে ।

✓ প্রোডাক্ট সার্চ ফিচার ইমপ্লিমেন্ট করতে হবে । সার্চের জন্যে API দিয়ে দেয়া হয়েছে। সেটি ব্যবহার করে সার্চ ফিচার ইমপ্লিমেন্ট করতে হবে।

✓ "Add to Cart" বাটনে ক্লিক করলে ওই প্রোডাক্ট টি কার্ট এ এড হবে এবং "Add to Cart" লিখাটি চেঞ্জ হয়ে "Remove From Cart" হয়ে যাবে। প্রোডাক্ট কার্ট এ এড করার জন্যে API বানিয়ে দেয়া হয়েছে । সেটি ব্যবহার করে Cart Manage করতে হবে ।

✓ কত গুলো প্রোডাক্ট কার্ট এ রয়েছে তার সংখ্যা Navbar এ Cart Link এর পাশে দেখাতে হবে ।

✓ "Remove From Cart" এ ক্লিক করলে কার্ট থেকে রিমুভ হয়ে যাবে

✓ Navbar এ Cart Link এ ক্লিক করলে আমাদের দেয়া 'cart.html' পেজ দেখাতে হবে । এখন পর্যন্ত যেহেতু আমরা Routing বা Page Navigation শিখায় নি, সেহেতু আপনাদের State দিয়ে Routing ম্যানেজ করতে হবে । যেমনঃ আপনি Application এর Root এ route নামে State ডিফাইন করতে পারেন । route এর ভ্যালু যদি হয় "cart" তাহলে cart পেজের রেন্ডার করবেন আর route এর ভ্যালু যদি হয় "home" তাহলে হোম পেজ রেন্ডার করবেন । অর্থাৎ Conditionally Render করে রাউটিং ম্যানেজ করবেন ।

✓ কার্ট পেজে সব গুলো Cart এ থাকা প্রোডাক্ট নিয়ে এসে দেখাতে হবে । সেই সাথে ডান পাশে থাকা Order Summary সেকশনে ব্যালেন্স আপডেট করতে হবে ।

✓ কার্ট পেজে থাকা প্রোডাক্ট গুলোর Quantity পরিবর্তন করা যাবে । তবে খেয়াল রাখতে হবে কোনো ভাবেই Stock এর থেকে বেশি পরিমাণ প্রোডাক্ট Quantity সিলেক্ট করা যাবে না । এর জন্যে সংশ্লিষ্ট API ব্যবহার করতে হবে ।

✓ কার্ট এ থাকা প্রোডাক্ট গুলো কার্ট থেকে রিমুভ করা যাবে । প্রতিটি প্রোডাক্ট এর সাথে রিমুভ বাটন রয়েছে সেখানে ক্লিক করলে কার্ট থেকে প্রোডাক্ট রিমুভ করে দিতে হবে । এর জন্যে সংশ্লিষ্ট API ব্যবহার করতে হবে ।
